About Me

My name is Sunny Nishad, a passionate graphic designer from Dongargarh, Chhattisgarh. I completed my schooling and graduation from Paonta Sahib, Himachal Pradesh. Initially enrolled in BCA with expectations towards technology, I discovered my true calling in the field of graphic design. I have gained hands-on experience in tools like Canva, Adobe Photoshop, Illustrator, and CorelDRAW. I'm keen on learning advanced tools like After Effects and Blender professionally.

Due to limited resources, I’ve worked mostly on lightweight tools, but my dedication and creativity have driven me to consistently upskill. I aspire to work in a metro city environment where collaboration, creativity, and professional growth are valued. I'm a quick learner, a strong team player, and constantly push myself toward excellence.

Projects

Solenoid - Your Store, Your Way (E-Commerce Website)

An online store prototype built to simulate real-time product listing and categorization. This is a homepage design for a conceptual e-commerce website named Solenoid, created using HTML and CSS only. Inspired by modern online shopping platforms, the layout features a functional header with navigation, search, cart sections, and footer, followed by promotional banners and categorized product previews. The project was built as a UI/UX design practice while following YouTube tutorials, focusing on clean structure, responsiveness, and visual appeal.


Shoe Store - Step Into Style Website

A conceptual e-commerce website focused on footwear branding and product visualization. This is a concept-based e-commerce website homepage focused on modern shoe branding and visual appeal. Built using HTML and CSS, this project emphasizes product-centric layout, clean UI, and stylish presentation.
The homepage features eye-catching banners, brand sections, and a neatly structured product grid. AI tools were used for layout planning and code reference, but the entire structure, styling, and customization were done manually, reflecting my creativity and practical design understanding.

Key Highlights:

  • Stylish, modern layout for shoe products
  • Promotional banner & grid-style product display
  • Minimalistic, clean, and visually attractive design
  • Developed with: HTML, CSS, and AI-assisted planning


Clothify Clothing - Style Made Simple Website

A user-friendly web layout for a clothing brand with clean UI and product displays. Clothify is a creative homepage design for a modern clothing brand, built using HTML and CSS. This project showcases a clean, user-friendly interface with a focus on product display, fashion appeal, and intuitive layout.
The layout includes featured banners, category blocks, and neat product sections. Design reference was planned with the help of AI tools, but all coding and customization were done manually to enhance skill development and real-world practice.

Key Highlights:

  • Minimal and aesthetic fashion layout
  • Dedicated sections for featured collections
  • Smooth UI design ideal for online clothing stores
  • Developed with: HTML, CSS, and AI-based layout references
